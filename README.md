# AEGIS – Adaptive Electrical Guardian for Intelligent Surveillance

## Project Overview
AEGIS is an innovative IoT-based embedded system designed for intelligent electrical monitoring and preventive protection using a Silicon Labs platform. The project focuses on real-time monitoring of key electrical parameters to identify abnormal behavior and enable early fault awareness, aiming to improve safety, reliability, and smart energy management.

## Problem Statement
Traditional electrical protection systems react only after faults occur. AEGIS aims to monitor electrical conditions intelligently and identify abnormal patterns for early fault awareness and preventive protection.
